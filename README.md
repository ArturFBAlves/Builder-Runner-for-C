# C Runner

C Runner is a simple tool designed to automate the compilation and execution of C programs. The script scans the project directory for `.c` files, compiles them using GCC, and runs the generated executable automatically.

The goal of this project is to make testing and running C programs faster and easier, especially during development.

## Current Features

* Automatically finds and compiles `.c` source files
* Builds the executable into an output directory
* Runs the compiled program after a successful build
* Allows passing additional arguments to the GCC compiler

## Future Features

* Automatic creation of C project structures
* Support for project templates
* Improved build configuration
* Additional development utilities

This project aims to become a lightweight tool to help developers quickly start and manage C projects.
